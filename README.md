**SnapMyFood 🍲📊**

SnapMyFood is an advanced, AI-driven dietary assistant designed to provide instant culinary and nutritional insights through image recognition. By combining Computer Vision with a comprehensive nutritional database, the platform allows users to simply capture or upload a photo of a dish to receive a detailed breakdown of its ingredients, cultural background, and macro-nutritional profile.

** ✨ Key Features**
Real-Time Dish Identification: High-precision recognition of diverse food items and traditional dishes using Deep Learning.

**Comprehensive Nutritional Profiles:**
Automatically calculates and displays Calories, Protein, Carbs, Fats, Fiber, and Sugar content per 100g.

Detailed Culinary Data: Provides essential information including ingredient lists, flavor profiles, preparation/cooking times, and the regional origin of the dish.

Live Camera Integration: Supports instant analysis via real-time camera capture as well as traditional file uploads.

User Analytics & History: Features a persistent history log that allows users to track their past analyses over time.

Personalized Dashboard: Includes secure user authentication with editable profiles and a modern, responsive UI.

**🛠️ Technology Stack**
Core Logic: Python & Flask

Artificial Intelligence: TensorFlow/Keras (CNN) & OpenCV for image processing.

Frontend Architecture: HTML5, CSS3 (featuring Glassmorphism design), and JavaScript.

Data Management: SQL-based database for managing user profiles and history logs.
