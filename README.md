# SnapMyFood 🍲📊

**SnapMyFood** is an advanced, AI-driven dietary assistant designed to provide instant culinary and nutritional insights through image recognition. By combining Computer Vision with a comprehensive nutritional database, the platform allows users to simply capture or upload a photo of a dish to receive a detailed breakdown of its ingredients, cultural background, and macro-nutritional profile.

## ✨ Key Features

* **Real-Time Dish Identification:** High-precision recognition of diverse food items and traditional dishes using Deep Learning models (CNN), providing instant classification with a visual confidence score.
* **Comprehensive Nutritional Profiles:** Automatically calculates and displays Calories, Protein, Carbs, Fats, Fiber, and Sugar content per 100g.
* **Detailed Culinary Data:** Provides essential information including ingredient lists, flavor profiles, preparation/cooking times, and the regional origin of the dish.
* **Live Camera Integration:** Supports instant analysis via real-time camera capture for on-the-go food logging, alongside traditional file uploads.
* **User Analytics & History:** Features a persistent history log that allows users to track and review their past food analyses over time.
* **Personalized Dashboard:** Includes a secure user authentication system with editable profiles and a modern, responsive Glassmorphism-based UI.

## 🛠️ Technology Stack

* **Backend:** Python & Flask
* **Artificial Intelligence:** TensorFlow/Keras (CNN) for classification & OpenCV for image processing.
* **Frontend:** HTML5, CSS3 (Modern Glassmorphism), and JavaScript.
* **Database:** SQL-based management for user accounts and historical analysis data.
